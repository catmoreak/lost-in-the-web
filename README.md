# Lost in the Web

A puzzle game built with Next.js where players navigate through challenges to escape the digital maze.

## What's this?


Just a fun little web game with some tricky puzzles. You'll need to solve challenges, find hidden flags, and race against others on the leaderboard. Think of it as an escape room, but on the web.

I have added 4 challenges as of now .Happy Bugging the flags !!
